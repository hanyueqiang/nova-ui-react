# Introduction

**Nova UI** is a modern, highly customizable AI UI component library for React. 

It provides out-of-the-box components for building conversational AI interfaces, including markdown-supported message bubbles, auto-resizing input areas, and typing effects.

## Why Nova UI?

Building a polished AI chat interface requires handling many small details:
- Rendering Markdown safely and beautifully.
- Syntax highlighting for code blocks.
- Auto-resizing textareas that support shortcuts (Enter to send, Shift+Enter to newline).
- Streaming text effects and cursor animations.

Nova UI abstracts all these complexities into a set of clean, accessible React components.
